# Claude Project Rules — Aman Verma Portfolio

## Tech Stack
- React 18 + TypeScript (strict)
- Vite (NOT CRA)
- Redux Toolkit for global state
- Framer Motion for animations
- Tailwind CSS v3

## Rules
- NEVER use deprecated packages (react-scripts, node-sass, moment, react-spring)
- ALL components must be typed with TypeScript interfaces
- Use functional components only — no class components
- Use Redux Toolkit `createSlice` — no legacy `combineReducers`
- Framer Motion `whileInView` for scroll animations
- Mobile-first CSS with Tailwind breakpoints
- Keep bundle size lean — no heavy unused libraries
- Accessibility: semantic HTML, ARIA labels on interactive elements
- No inline styles — use Tailwind classes or CSS variables

## File Conventions
- Components: PascalCase.tsx
- Hooks: use[Name].ts
- Types: PascalCase.ts in /types
- Data: camelCase.ts in /data
